# skydiving-frontend
Frontend for [Skydiving LINE BOT](https://github.com/hcmkt/skydiving)

## Development
Copy `.env`
```
cp .env.example .env
```
Install packages
```
yarn
```
Start dev server
```
yarn dev
```

### Mock server
Initialize settings
```
task init
```
Start JSON Server
```
task json-server
```
